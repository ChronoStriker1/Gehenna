# Requirements

## V1 Scope (Must Have)
- Detect and connect to Razer Tartarus Pro via HID.
- Key remap for all buttons.
- Macro recording.
- Macro editor (sequence editing, insert/delete).
- Macro delays (fixed and variable).
- Profiles (global and per-app).
- Import/export of profiles.

## Non-Goals (V1)
- Cloud sync.
- Community macro sharing.
- Firmware updates.

## Quality
- Native macOS UI and behavior.
- High security posture and least-privilege architecture.
- Strict formatting and linting (SwiftFormat + SwiftLint).
- Stable on macOS 26.2.
