# Roadmap

## Phase 0: Research & Device Discovery
- Confirm USB vendor/product IDs.
- Validate HID usage pages and report formats.
- Prototype event read/write.

## Phase 1: Core Service
- Daemon skeleton + LaunchAgent.
- XPC interface.
- HID connection and event stream.

## Phase 2: Macro Engine
- Macro data model.
- Recorder and editor logic.
- Execution engine with delays.

## Phase 3: UI
- Device status screen.
- Profile management.
- Macro editor UI.

## Phase 4: Hardening
- Permissions flows.
- Logging, crash handling.
- QA and stability.
